### Descrição

Implemente o módulo MCP, permitindo o gerenciamento de eventos de memória local para cada projeto.

### Tarefas
- [ ] Configurar `.cursor-memory/memory.json` como local padrão para gravação de eventos.
- [ ] Implementar `memory.add` para adicionar eventos no arquivo JSON.
- [ ] Implementar `memory.list` para retornar todos os eventos.
- [ ] Implementar `memory.clear` para apagar eventos existentes.

### Critérios de aceitação
- Métodos devem seguir a especificação do projeto fornecida.
- Testes básicos para cada método com exemplos claros.
